# sonnytaylor.me

Personal portfolio site. Built with Next.js 16, React 19, and Tailwind CSS v4.

**Live at [sonnytaylor.me](https://sonnytaylor.me)**

## Highlights

- **Workbench hero** — parallax tools with directional shadows, depth-of-field blur, and a desk lamp light simulation. Coffee stains and scratches included.
- **Interactive project cards** — each featured project has a themed animated visual panel (network topology, service queues, CV classification, extraction demos, dashboard mockups)
- **3D tilt on hover** — perspective transforms track cursor position across project cards
- **Terminal animation** — step-based Claude Code terminal with custom color markup
- **PCB + e-ink experience section** — circuit board traces, LED indicators, and e-ink display modules
- **Corkboard certifications** — draggable pinned cards with issuer logos and thermal receipt

## Stack

Next.js 16 (App Router) · React 19 · TypeScript · Tailwind CSS v4 · motion/react · shadcn/ui

## Development

```bash
bun install
bun dev
```
